首先非常欢迎你选择了 Ponder, 接下来让我们回答一些常见的问题

Ponder 是什么?

  • Ponder 是由 Create 团队研发的一个用于制作交互式游戏内指南的库, 旨在帮助玩家理解模组新增的内容.
  • 借助 Ponder, 模组作者可以精心设计细致入微且引人入胜的教程, 以直观的方式展示模组机制, 让玩家更轻松地掌握模组的使用方法.
  • 在此前, Ponder 一直作为 Create 的内置功能存在.
  • 在 Create 更新至 6.0 版本后, Ponder 被拆分为独立的 Lib Mod, 这意味着开发者无需依赖Create本体, 也可以为自己的模组编写独立的 Ponder 教程.

为什么选择 Ponder

  • 相比于传统的文本说明书和全是文字的wiki, Ponder 最大的优势在于:
    • 动画和结构的动态展示更为直观
    • 更适合一些多方块结构的的搭建和使用教学
    • 更复杂的机械结构展示
    • 能够直接在游戏内完成教学

该教程支持的是什么版本

  • 该教程只会支持 Create 在 6.0 版本分离后的单独的版本, 分离前的版本并不支持

在学习 Ponder 前需要准备什么条件

  • 一些非常非常基础的 Java 开发能力
  • 基础的 Minecraft (Neo)Forge Mod 的开发经验
  • 对 Block, BlockState, Item, ItemStack, BlockEntity, NBT(Component) 等内容有一定的了解

教程中使用的环境是什么

  • Minecraft-1.20.1
  • Forge-47.4.0
  • Java-17
  • Ponder-1.0.91
  • Flywheel-1.0.5
  • IntelliJ IDEA
